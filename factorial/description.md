# Factorial

Write a function that returns the factorial of the number 'n'.  For example, the factorial of 4 is '4 * 3 * 2 * 1 = 24'

Example 1:

```js
factorial(1); // => 1
```

Example 2:

```js
factorial(4); // => 24
```

Example 3:

'''js
factorial(5); // => 120
'''

Example 4:

'''js
factorial(10); // => 3628800
'''
