# Array Range

### High level idea

Construct a for loop with i starting at min, iterating at i + step until i <= max, pushing in values of i into a return array.  

### Time and space complexity

Let: <br>

- `n` be the (max - min)/step <br>

Worst cases: <br>

- Time: `O(n)` <br>
- Space: `O(n)`
