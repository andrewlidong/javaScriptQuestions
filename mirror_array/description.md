# Mirror Array

Write a function mirrorArray that takes in an array as an argument and returns a new array "mirrored" as shown in the examples.  

Examples:

```js
mirrorArray([1,2,3]); // => [ 1, 2, 3, 3, 2, 1 ]
mirrorArray(['a', 'b', 'c', 'd']); // => [ 'a', 'b', 'c', 'd', 'd', 'c', 'b', 'a' ]
```
