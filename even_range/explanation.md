# Even Range

### High level idea

Iterate through from start to end, checking if i modulo 2 is 0, and push if so.  

### Time and space complexity

Let: <br>

- `n` be the size of the input array <br>

Worst cases: <br>

- Time: `O(end-start)` <br>
- Space: `O(end-start/2)`
