# Object Size

Write a function objectSize(obj) that takes in an object and returns the number of key-value pairs in that object.  

Examples:

```js
const obj1 = {
  name: 'Fido',
  age: '4'
}

objectSize(obj1); // => 2

const obj2 = {
  name: 'App Academy',
  course: 'Bootcamp Prep',
  locations: ['SF', 'NY']
}

objectSize(obj2); // => 3```
