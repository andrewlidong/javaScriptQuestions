# First or Last

### High level idea

Check whether the array length is odd or even.  If it's even return the first element, otherwise return the last.  

### Time and space complexity

Let: <br>

- `n` be the size of the input array <br>

Worst cases: <br>

- Time: `O(n)` <br>
- Space: `O(1)`
