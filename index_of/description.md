# Is Inside

Write a function isInside(array, el) that returns true if el is an element inside of the 'array', false otherwise.  

Example 1:

```js
isInside(['chase', 'phi', 'eliot'], 'eliot'); // => true
```

Example 2:

```js
isInside(['chase', 'phi', 'eliot'], 'fred'); // => false
```

Hint:

Use indexOf
